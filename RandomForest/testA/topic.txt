📌 题目：利用随机森林预测房屋是否能在 30 天内卖出
背景设定

你现在是一家房产中介的数据分析师。公司给你一份历史房屋交易数据，想让你构建一个模型去预测：
这套房子是否能在挂牌后 30 天内售出（二分类）。

公司希望你给出：

一个性能良好的随机森林分类模型；

特征重要性分析，告诉他们“哪些因素影响房子的快速售出”；

模型泛化能力评估（包含 OOB 误差或交叉验证）；

解释随机森林为什么适合这个任务。

📜 提供给你的数据字段（假设有约 3000 条记录）
字段	                类型	说明
price	            数值	挂牌价格（万元）
area	            数值	房屋面积
bedrooms	        数值	卧室数
bathrooms	        数值	卫生间数
year_built	        数值	建造年份
renovated	        0/1	是否翻修
distance_to_metro	数值	距离地铁站（公里）
community_score	    数值	小区评分（1–10）
days_on_market	    数值	实际售出所用天数（目标字段将基于它生成）
目标变量（你需要自己构造）
sold_fast = 1 if days_on_market <= 30 else 0

🎯 任务要求
1. 数据预处理

构造目标变量 sold_fast

划分训练集 / 测试集

optional：简单探索数据（缺失值、分布等）

2. 训练随机森林模型

尝试不同的超参数：
n_estimators, max_depth, max_features, min_samples_split

使用 OOB 误差 或 5-fold CV 来选择最佳参数

评估模型（accuracy, precision, recall, F1）

3. 特征重要性分析

要求你给出两种：

基于 Gini 的 importance（树模型自带）

基于 permutation importance（置换）

并解释差异。

4. 可解释性输出（英文也可以）

公司希望你给出结论，包括：

哪些特征最能决定房屋能否快速卖出

特征重要性的业务含义

模型的可靠性（泛化能力如何）

随机森林为什么适合用于这种类型的任务（强鲁棒性、非线性、对噪声不敏感等）

🧩 你需要写 Python 完成整个任务

允许你使用：

pandas / numpy

scikit-learn

matplotlib（如需可视化）

最终你要交付：

模型训练代码

模型性能指标

特征重要性图

模型总结（书面解释）
