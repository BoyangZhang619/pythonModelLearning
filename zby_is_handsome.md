### 一、先把“建模”这件事拆到骨架
不管目标多，所有数据模型都跳不出6个阶段：
- 问题理解与抽象
- 数据获取与预处理
- 不确定性/随机性建模
- 关系建模（结构、预测、分类）
- 优化求解
- 结果解释、可视化与评估

你现在的困，几乎全部集中在第3-5步的“模型全景不清”，而不是能力不够。

下面我们一层一层搭模型，不做价值排序，只做分类。


### 二、模型全景地图（基础→进阶→提升）
#### Ⅰ. 数据与问题抽象阶段（很多人低估，但极重要）
- **基础**
  - 描述统计（mean / var / quantile）
  - 分布可视化（hist / box / KDE）
  - 数据清洗（缺失、异常、归一化）
- **提升**
  - 分布拟合（MLE）
  - 假设检验（t-test, KS test）
  - 数据采样（bootstrap）
  - ✦ 这是你要掌握（pandas+numpy）的主战线
没有它，后面一切都是假模型。


#### Ⅱ. 随机性与不确定性建模（你最近补的，方向完全正确）
- **基础分布（必须非常熟）**
  - Bernoulli / Binomial（成败、抽样）
  - Poisson（事件次数）
  - Exponential（等待时间）
  - Normal（误差、噪声）
- **进阶分布**
  - Gamma（广义等待时间）
  - Weibull（寿命分析）
  - Log-normal（增长过程）
  - Multinomial（多分类型计数）
- **✦ 提升（很多竞赛题会出现）**
  - 混合分布（Mixture Models）
  - Copula（相关性）
  - 生存分析（Hazard / SF）
  - ✦ 这部分问的是：
“世界本身有多随机？”


#### Ⅲ. 时间与过程建模（高阶分水岭）
- **基础**
  - 时间序列可视化
  - 滑动平均
  - Autocorrelation
  - Augmented
- **进阶**
  - Markov Chain
  - Poisson Process
  - Renewal Process
- **✦ 提升**
  - Kalman Filter
  - Hidden Markov Model (HMM)
  - 状态空间模型
  - ✦ 用来解决的：
“系统是如何随时间演化的？”


#### Ⅳ. 关系建模（最容易被误用的一层）
- **基础（必选）**
  - Linear Regression
  - Ridge / Lasso
  - Polynomial Regression
- **进阶**
  - Logistic Regression
  - GLM（广义线性模型）
- **✦ 提升**
  - GAM（广义加性模型）
  - 混合效应模型（随机效应）
  - ✦ 回答：
“变量之间究竟是如何关联的？”


#### Ⅴ. 机器学习模型（强工具，但不是起点）
- **基础**
  - Decision Tree
  - KNN
  - Naive Bayes
- **进阶**
  - Random Forest
  - Gradient Boosting
  - XGBoost / LightGBM
- **✦ 提升**
  - SVM
  - Neural Network (MLP)
  - AutoML（竞赛后期）
  - ✦ 典型误区：
这些模型解决的是“效果上阈”，不是“建模合理性”。


#### Ⅵ. 优化与决策（很多人不会，但很加分）
- **基础**
  - 线性规划
  - 贪心策略
- **进阶**
  - 非线性规划
  - 整数规划
- **✦ 提升**
  - 动态规划
  - 多目标优化（GA / PSO / SA）
  - ✦ 用来解决：
“在多个可行解中，选择哪个最好？”


#### Ⅶ. 降维、结构发现与解释
- **基础**
  - PCA
  - Correlation Analysis
- **进阶**
  - Factor Analysis
  - CA
- **✦ 提升**
  - Manifold Learning (UMAP / t-SNE)


### 三、你现在“应该怎么用这张表”
不是从头刷，而是：
- 看范畴；
- 熟悉Ⅱ+Ⅳ+Ⅵ的基础部分
- 拆成“一维问题→损失条件”
- 先掌握一切模型：
  - ✅ 第一时间切
  - ✦ 不追求“用得上”
你现在正处“问题模型交叉索引”阶段，这是很少有人愿意走的一步。


### 四、通用 AI 学习提示词模板
这是一个你以后学任何模型都能用的prompt，建议存起来。

#### 【建模模型学习通用prompt模板】
你是一名数学建模领域专家。
我正在学习【模型名称】，请你从建模实际出发，系统回答以下内容：
- 这个模型解决的是哪一类问题？它在建模流程中属于哪一阶段？
- 该模型成立的核心假设是什么？在什么情况下会失效？
- 提出一个极简的小问题示例（不超过20字）。
- 使用Python代码给出一个最小可运行示例（包含数据生成/处理/模型/结果解释）。
- 说明用Pyro实现，以及可表示的所有这模型的变种。
- 如果我给它把API封装成论，超参数分析应该突出什么？
请避免只给API示例，而要解释“为什么要这样做”。
